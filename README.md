<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* Java Development Kit (JDK)
* Java Runtime Environment (JRE)
 
### Installation

1. Clone the repo into a directory you want.
2. Go to the 'src' folder
3. Open up a terminal in folder
4. Type
   ```
   javac Game.java
   ```
5. Start playing by typing
   ```
   java Game
   ```
