## Getting Started
### Installation

1. Clone the repo
   ```
   git clone https://github.com/baranacikgoz/dynamic-tic-tac-toe-game.git
   ```
2. Go to the folder "src"
3. Open up a terminal in folder
4. Type ``` javac Game.java ```
5. Start playing by typing ``` java Game```
